**Date:** February 8, 2026

## 1. Executive Summary
Nextaro is a high-precision, AI-driven career intelligence platform designed to automate professional mastery. It provides users with a centralized "neural command center" to track career progress, discover learning paths through AI recommendations, and exchange skills within a global professional engine.

## 2. Technical Stack
### **Frontend (The Interface)**
- **Framework:** React.js (v19) via Vite
- **Styling:** Tailwind CSS (Vanilla CSS focus)
- **UI Components:** Radix UI primitives, Lucide React (Icons), Sonner (Toasts)
- **Animations:** Framer Motion for smooth transitions and hover effects
- **Routing:** React Router DOM (v7)
- **API Client:** Axios for robust HTTP communication

### **Backend (The Engine)**
- **Environment:** Node.js
- **Server:** Express.js
- **Security:** JWT (JSON Web Tokens), Bcryptjs (Password hashing), Helmet, HPP, XSS-Clean
- **Logging:** Morgan

### **Database (Persistence Layer)**
- **DB:** MongoDB (Cloud Atlas Cluster)
- **ORM:** Mongoose

### **AI Services (Intelligence Layer)**
- **Google Generative AI:** Gemini-2.0-flash
- **Groq SDK:** High-speed Llama models
- **OpenAI:** GPT-4o for complex reasoning
